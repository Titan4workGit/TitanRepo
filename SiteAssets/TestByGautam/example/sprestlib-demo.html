File Manipulation: (Delete/Recycle, Download)

Download File
Saving a SharePoint file (text or binary) using client web browser.
The `sprLib` Promise returns a blob which can be saved easily - without FileSaver or other libraries. 
sprLib.file('fileName').get()
Delete File (PERMANENT - bypasses Recycle Bin) 
sprLib.file('fileName').delete()
Recycle File (sends item to the site Recycle Bin) 
sprLib.file('fileName').recycle()
