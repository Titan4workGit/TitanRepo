Instructions

  • The file must be Excel Workbook (*.xlsx) or Excel 97-2003 Workbook (*.xls)
  • File must content raw data without any formatting, grouping of columns/rows or additional sheet.
  • The first row is a header line. System will skip this header line to import process.
  • The data must be in given sequence, not allowed to misplaced any column.
  • The maximum limit of rows in one Excel file is 5000.
  • The * marked columns are mandatory.
  • Company, Department and Office Location, must be ID not the Name or title.
  • Collect those IDs form Admin Panel corresponding section like Manage Company, Manage Department, Manage Location.
  • Users, Managers are identified by email address of their O365 AD profiles.
  • If Manager information is missing then that user treated as ‘Top Management’.
  • The date format of DOJ (Date of Joining), DOB (Date of Birth) and Anniversary must be MM/DD/YYYY.
  • Define properly users-type (Internal or external). All external user must be pre-invited in the tenant and available in O365 profile.
  • Char limit for Designation, Group Name, Mobile, City, State, PIN, Email, Country are 250.
  • Char limit of current and permanent address are 500.
  • All data must be in English.
